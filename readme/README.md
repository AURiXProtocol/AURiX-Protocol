# AURiX-Protocol
Invariant Bound Attestation Benefit

AURiX is a protocol, not a product.

It observes structure, not people.
It produces posture, not verdicts.
Correctness is determined by replay, not authority.

Core primitives:
- LPIN (Local · Private · Invariant · Non-extractive)
- Replay
- Posture (Stable / Uncertain / Unstable)
- Quadratic Context (Space · Time · Path · Meaning)
- Constraint / Boundary

This repository contains the canonical specification, library,
and teaching surfaces for AURiX.
