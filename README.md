# AURiX Protocol — Browser Artifact (Early)

This repository contains an early, inspectable browser-based artifact related to the AURiX protocol.

The contents here are intentionally limited in scope and focus on observable browser interaction behavior rather than protocol explanation or claims.

## What this is
- A small, local-only artifact for evaluation
- A concrete surface that demonstrates behavior

## What this is not
- A complete protocol specification
- A finished product
- A roadmap, pitch, or positioning document

## Notes
The protocol itself exists beyond this repository.
This repo holds only what is necessary to inspect the artifact in isolation.

Local-only.  
No data collection.  
No network requests.
